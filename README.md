# ポテトくん VRM配布サイト 🎄

フライドポテトをモチーフにしたオリジナルVRMアバター「ポテトくん」の無料配布サイトです。

## 🎅 クリスマス特別仕様

現在、サイト全体がクリスマス仕様になっています!
- ❄️ 降る雪のアニメーション
- 🎄 クリスマスカラーのライティング
- ✨ 特別な装飾とエフェクト

## 🌐 サイトURL

https://nine-jp.github.io/potatokun/

## 📦 含まれるファイル

- `index.html` - メインページ
- `style.css` - スタイルシート
- `script.js` - 3Dビューアーのスクリプト
- `uma2025-6.fbx` - 3Dモデル

## 🎨 技術スタック

- HTML5
- CSS3 (アニメーション、グラデーション)
- JavaScript (ES6 Modules)
- Three.js (3Dモデル表示)

## 📝 ライセンス

- 個人利用OK
- 改変OK
- 再配布NG
- 公序良俗に反する利用NG
